# 10d — aide: custom assistants

ROADMAP 10d: personas + knowledge files + share (1a) + more MCP connectors (one-click presets).
Acceptance: a persona answers from its attached files; a shared bundle imports; an MCP preset connects
in one click.

## Audit — `docs/evidence/10d/findings.md`
Personas exist (system_prompt/model/temp/mode), applied in chat.py. No knowledge files. 1a share +
`/s/{token}` dispatch exists (add a `persona` kind). MCP servers stored in `McpServer`; no presets. 1c
text index reused with kind `persona:<id>`.

## 10d-1 — persona knowledge files (unittest) — `tests/test_persona_docs.py`
Files: `core/database.py` (`PersonaDoc`), new `services/persona_docs.py`, `routes/personas.py`,
`routes/chat.py` (inject knowledge into the system prompt).
- `PersonaDoc(id, persona_id, title, created_at)`; content lives in the 1c index under
  `kind=f"persona:{pid}"`, `ref=doc.id`.
- `services/persona_docs.py`: `attach(db,pid,title,content)`, `list_docs(db,pid)`,
  `detach(db,pid,doc_id)`, `purge(db,pid)`, `knowledge_block(db,pid,query,k=5)` (top chunks joined).
- Endpoints: `POST/GET /api/personas/{pid}/docs`, `DELETE /api/personas/{pid}/docs/{doc_id}`.
- chat.py: when a persona is active, prepend its `knowledge_block(user_message)` to the system prompt;
  deleting a persona purges its docs + index chunks.
Tests (≥8): attach_creates_and_indexes, list_docs, detach_removes_doc_and_chunks, knowledge_block_matches,
knowledge_block_empty_no_docs, cross_persona_isolation, purge_clears, api_attach_list_detach,
delete_persona_purges_docs, knowledge_block_empty_no_match.

## 10d-2 — persona share + MCP presets (unittest) — `tests/test_assistants_share_mcp.py`
Files: `services/share.py` (add `persona` kind), `routes/shared.py` (persona bundle page),
`routes/personas.py` (`POST/DELETE /api/personas/{pid}/share`), `routes/mcp.py` (`MCP_PRESETS` +
`GET /api/mcp/presets` + `POST /api/mcp/presets/{id}`).
- Share a persona via 1a; `/s/{token}` renders a public bundle (name/emoji/prompt/model/initial msg +
  attached knowledge titles) with an "add to my aide" affordance.
- MCP presets: a curated static list; one POST creates (and tries to connect) the server with args
  interpolated from params.
Tests (≥8): mint_persona_kind, share_endpoint_mints, view_persona_bundle_html, revoked_persona_404,
presets_nonempty, preset_schema, add_preset_creates_server, add_preset_unknown_404,
add_preset_interpolates_params, share_persona_unknown_404.
