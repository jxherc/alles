# 8c — contacts depth

ROADMAP 8c: multiple labeled fields + social/custom, photo/avatar, groups w/ smart membership,
dup-detect/merge/linked, Me card, address map.

## Audit (current state)
- `Contact` is flat: single `email`/`phone`/`address`, plus company/title/birthday/website/tags/favorite.
- No labeled multi-fields, no avatar, no groups, no duplicate-merge, no Me card.
- vCard import/export exists; address map is just a link (trivial, frontend-only).

## 8c-1 — multi labeled fields + avatar + Me card (unittest) — `tests/test_contacts_fields.py`
Files: `core/database.py` (`ContactField` model + `Contact.avatar`, `Contact.is_me`),
`routes/contacts.py`, `routes/uploads.py` reuse for avatar storage.
- `ContactField`: id, contact_id, kind (email|phone|address|url|social|custom), label, value, sort_order.
- `GET/POST/DELETE /contacts/{cid}/fields`; `_fmt` carries `fields` + `avatar` + `is_me`.
- avatar: `POST /contacts/{cid}/avatar` (image upload → stored), `GET /contacts/{cid}/avatar`, DELETE.
- Me card: `PATCH is_me=true` clears it on others (single Me); `GET /contacts/me`.
Tests (>=8): add_field, field_kinds, list_fields_in_fmt, delete_field, set_avatar, get_avatar,
avatar_unknown_404, set_me_card, me_card_is_singular, get_me, custom_field_label.

## 8c-2 — groups + duplicate detect/merge (unittest) — `tests/test_contacts_groups.py`
Files: `core/database.py` (`ContactGroup`, `ContactGroupMember`), `routes/contacts.py`.
- Groups: manual (explicit members) or smart (auto by `rule_tag` / `rule_company`).
- `GET/POST/DELETE /contacts/groups`; `GET /contacts/groups/{gid}/members` (computed for smart);
  `POST/DELETE /contacts/groups/{gid}/members` (manual).
- `GET /contacts/duplicates` → clusters by normalized name + shared email.
- `POST /contacts/merge {primary_id, other_id}` → fold other's fields/tags/notes into primary, delete other.
Tests (>=8): create_group, manual_add_member, smart_group_by_tag, smart_group_by_company,
delete_group, duplicates_by_name, duplicates_by_email, no_duplicates_empty, merge_combines_fields,
merge_deletes_other, merge_unknown_404.
