# 6b — files: browse & dedup

ROADMAP 6b: grid thumbnail view, office-doc preview, exact-hash duplicate detection, activity view.
(Perceptual/visual dedup = out-of-scope; exact SHA-256 only.)

## 6b-1 — backend (unittest)
Files: `routes/files.py`.
- `GET /api/files/duplicates` — SHA-256 every file under the vault, group hashes with >1 file
  (skip empty). Returns `{groups:[{hash, size, paths:[...]}]}`.
- `GET /api/files/preview?path=` — docx → extracted text/HTML (python-docx, installed); xlsx → graceful
  "openpyxl not installed"; txt/md → text; else `{kind:"unsupported"}`. 404 for a missing file.
- `GET /api/files/activity?days=&limit=` — files changed in the last N days, mtime-desc
  (`[{path, mtime, size}]`).

Tests (`tests/test_files_browse.py`, ≥8):
1. `dup_groups_identical` 2. `dup_ignores_singletons` 3. `dup_ignores_empty`
4. `preview_docx_text` 5. `preview_txt` 6. `preview_unsupported` 7. `preview_missing_404`
8. `activity_recent_first` 9. `activity_excludes_old`

## 6b-2 — frontend (Playwright)
