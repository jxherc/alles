non-sub bill reminders + low-balance alerts, cancel-helper.

## 4e-1 — backend (unittest)
Files: `core/database.py`, `routes/subscriptions.py`, `routes/money.py`.
- `Subscription.cancel_url` column (the explicit "how to cancel" link, distinct from `url`);
  `_fmt`/`SubBody`/`SubPatch`/create/update handle it.
- `GET /api/subscriptions/unused?cycles=N&as_of=` — flag active subs with no matching money charge
  (sub name in payee/notes/category) within the last N cycle-lengths.
- `Account.low_balance` threshold column; `_acct` exposes it; `AccountBody`/PATCH accept it.
- money `GET /api/money/alerts` gains a `low_balance` list (accounts whose balance < threshold).
