sidebar becomes an off-canvas overlay drawer with a dim backdrop (auto-hidden by default on
phone), tap backdrop / nav item closes it; topbar toggle opens it. Guard horizontal overflow,
clamp modal widths to viewport, bump tap targets. Files: `static/style.css`, small `app.js` hook
to close drawer on nav + add backdrop element, `index.html`.
Tests (`tests/pw_mobile_11b.py`, 390x844): sidebar_hidden_by_default, toggle_opens_drawer,
backdrop_present_when_open, navitem_closes_drawer, no_horizontal_overflow, modal_fits_viewport,
navitems_tappable_height, topbar_visible, zero_console_errors.

### 11b-3 — Offline app-shell pre-cache (cold offline)
SW only network-first caches as-you-go → a cold offline load (never visited) fails. Add an
install-time `precache` of the shell (`/`, `/static/style.css?v=N`, app.js, manifest, icons,
key JS modules) so the app boots offline on first run. Bump SW VERSION + asset stamp.
Files: `static/sw.js`, `static/index.html` (stamp), bump.
Tests (`tests/pw_offline_11b.py`): shell_precached_on_install, cold_offline_boots,
offline_edit_persists, pending_indicator_shows, reconnect_flushes, sw_version_bumped,
precache_list_nonempty, static_assets_cached, zero_console_errors.

`mobile/capacitor.config.json` (server.url → the alles host, so the shell loads the live PWA),
`mobile/package.json` (capacitor deps + add-platform scripts), `mobile/README.md` (build steps
for iOS/Android), `mobile/www/index.html` (redirect splash). Honest seam like 11a's macOS bridge.
Tests (`tests/test_capacitor_11b.py`): mobile_dir_exists, config_valid_json, config_app_id,
config_server_url_configurable, package_json_valid, package_has_capacitor_deps,
package_has_platform_scripts, readme_mentions_build, www_index_present.
