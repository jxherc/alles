# 10e — aide: vision + browser

ROADMAP 10e: video-input understanding (frame extraction for capable models), browser-automation tool
(a Playwright/CDP agent tool distinct from pixel computer-use). Video *generation* stays out of scope.

## Audit
Image vision already works: `routes/chat.py _build_messages` turns image uploads into `image_url`
parts. Computer-use (pyautogui, pixel-level) exists in `agent_tools.py`; **no Playwright/CDP browser
tool**. ffmpeg is available (used by `services/photos_store.py` for video thumbnails). No video-frame
extraction for the model.

## 10e-1 — video-input understanding (unittest) — `tests/test_video_frames.py`
Files: new `services/video_frames.py`, `routes/chat.py` (video branch in the upload loop),
`static/index.html` (let the attach picker accept video).
- `services/video_frames.py`: `is_video(mime, name)`; `_sample_times(duration, n)` (evenly spaced,
  capped); `extract_frames(path, n=8) -> list[str]` (ffmpeg → JPEG frames → `data:image/jpeg;base64,…`
  URLs; `[]` if ffmpeg missing / file absent — never raises); `MAX_FRAMES`.
- `chat.py`: an uploaded `video/*` → `extract_frames` → append the frames as `image_url` parts so a
  vision model can describe/summarize the clip.
Tests (≥8): is_video_by_mime, is_video_by_ext, not_video, sample_times_even, sample_times_capped,
sample_times_short, extract_missing_file_empty, extract_no_ffmpeg_empty (monkeypatched), frame_data_url_shape
(monkeypatched runner), max_frames_enforced.

## 10e-2 — browser-automation tool (unittest) — `tests/test_browser_tool.py`
Files: new `services/browser_tool.py`, `services/agent_tools.py` (tool defs + dispatch).
- `services/browser_tool.py`: a lazily-launched headless Playwright session with
  `navigate(url)`, `read_text()`, `click(selector)`, `type_text(selector, text)`,
  `current_url()`, `screenshot()`, `close()`. Distinct from pixel computer-use — it drives the DOM.
- Agent tools `browse_open`, `browse_read`, `browse_click`, `browse_type`, `browse_screenshot`
  wired into `execute()` + `build_tool_defs` (+ MUTATING/relevant sets as needed).
Tests (≥8, drive real headless chromium against a temp `file://` page): navigate_and_read,
read_after_navigate, click_changes_dom, type_into_input, current_url, screenshot_returns_png_b64,
tool_browse_open, tool_browse_read, tool_browse_click, close_resets. (Skip cleanly with a clear message
only if Playwright/chromium is genuinely unavailable — it is installed here.)
