  (exists, **no UI**), quick, CRUD, `export.ics`, `import` (pasted ICS only — **no URL subscribe**).

## 8a-1 — ICS URL subscriptions (unittest) — `tests/test_calendar_subscriptions.py`
Files: `core/database.py` (`CalendarSubscription` model + `CalendarEvent.subscription_id`),
`routes/calendar.py`, `app.py` (refresh job).
- `CalendarSubscription`: id, name, url, calendar_id, last_synced, last_status.
- `refresh_subscription(db, sub, ics_text)` full-replaces that sub's events (delete where
  `subscription_id==sub.id`, re-insert parsed events into `sub.calendar_id`); sets last_synced/status.
  `fetch_ics(url)` is separate + injectable so tests don't hit the network.
- Endpoints: `GET/POST/DELETE /calendar/subscriptions`, `POST /calendar/subscriptions/{sid}/refresh`.
- Job `ics_subscriptions` refreshes all subs hourly.
Tests (>=8): create, refresh_imports, refresh_replaces_no_dupes, refresh_updates_changed, delete_removes
sub+events, list_with_counts, events_linked_to_calendar, last_synced_status_set, empty_ics_ok, fetch_error_status.
